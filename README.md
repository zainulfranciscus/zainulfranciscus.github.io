# **Project Portfolio**

Welcome to my project showcase\! This page documents examples that demonstrate intelligent workflows built using Agentic AI Coding tool called [Antigravity](https://antigravity.google/)

## **📂 Featured Projects**

| Project | Summary |
| :---- | :---- |
| **[1\. Intelligent Customer Support Agent](https://github.com/zainulfranciscus/customer-support-agent)** |  This project demonstrates an LLM that routes and answers incoming queries. When a user submits a question, the system first preprocesses and saves it, then uses an AI classifier to determine if the query is related to shipping (such as tracking, delivery, rates, or returns) or not. Depending on this classification, the system routes the request to either a specialized **shipping FAQ agent** to answer the question or a **decline agent** that politely explains the system can only assist with shipping-related topics. |
| **[2\. Serverless Event-Driven Document Pipeline](https://github.com/zainulfranciscus/google-cloud-serverless-app)** | This project is an automated document processing system built on Google Cloud Platform (GCP). When you upload a document (like a text file, image, or PDF) to cloud storage, the system automatically detects it, analyzes its contents (counting words, extracting text, and tagging it), and logs this information in a database. It also provides a web dashboard to view the processed documents. |
| **[3\. AI Agent built using Google Agent Development Kit ](https://github.com/zainulfranciscus/ambient-expense-agent)** |This project demonstrates how to build a decision making tool that uses FastAPI and Google Cloud Pub/Sub with the Agent Development Kit (ADK) 2.0. Its core features include a self-protecting security checkpoint that redacts PII and blocks prompt injections, conditional routing to a Gemini-powered audit agentor a native Human-in-the-Loop. This project comes with an LLM-as-a-judge evaluation framework that captures execution traces and grades the agent on routing correctness and security containment |
